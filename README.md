# San Diego Housing Analysis (Power BI)

This project analyzes housing data in San Diego to identify trends in prices, neighborhood statistics, and market patterns. 

## Tools Used
- Power BI

## Data Used
- Sample San Diego housing data  
- File: `san_diego_housing_sample.csv`

## What I Did
- Loaded and cleaned the housing dataset
- Created visualizations to show price trends, neighborhood comparisons, and key market metrics
- Designed an interactive Power BI dashboard highlighting trends and insights
- Included a dashboard snapshot for quick overview

## Key Insights
- Housing prices vary significantly between neighborhoods
- Certain areas showed increasing trends over time
- Visualizations help identify high-performing and undervalued neighborhoods
